HTML

HTML (HyperText Markup Language) é a linguagem de marcação que define a estrutura e organização dos elementos na página, como títulos, parágrafos, imagens e links. Ele forma a base do conteúdo da página.

CSS

CSS (Cascading Style Sheets) é a linguagem que controla o design e a apresentação visual da página, permitindo definir cores, fontes, margens e layout. Ele melhora a aparência e torna a página mais agradável aos olhos.

JS

JavaScript é uma linguagem de programação que permite adicionar interatividade à página. Ele pode ser usado para criar animações, responder a eventos do usuário, modificar o conteúdo em tempo real e se comunicar com servidores, tornando a página dinâmica e responsiva.
